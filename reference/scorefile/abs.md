## abs

Return absolute value of argument.

-----

SYNOPSIS

posval = **abs**(*val*)

-----

DESCRIPTION

**abs** returns the absolute value of its argument.

-----

ARGUMENTS

  - *val*  
    Any number, floating point or integer

-----

EXAMPLES

``` 
   posval = abs(arbitrary_val)
```

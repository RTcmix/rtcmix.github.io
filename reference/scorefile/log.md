## log

Return log<sub>10</sub> of the argument.

-----

### Synopsis

val = **log**(*someval*)

-----

### Description

**log** returns the log<sub>10</sub> of its argument.

-----

### Arguments

  - *someval*  
    Any number, floating point or integer

-----

### Examples

```cpp
   lval = log(3.414)
```

## exit

Terminate RTcmix.

-----

### Synopsis

**exit**()

-----

### Description

**exit** closes all open soundfiles and input streams (if any) and
terminates RTcmix execution. This happens immediately upon the parser
encountering the **exit** command &mdash; all scheduled events are
terminated.

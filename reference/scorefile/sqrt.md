## sqrt

Return the square root of the argument

-----

### Synopsis

val = **sqrt**(*number*)

-----

### Description

**sqrt** returns the square root of any non-negative argument.

-----

### Arguments

  - <span id="item_base">*number*</span>  
      
    Any non-negative number. 

-----

### Examples

```cpp
val = sqrt(3.5)
val = sqrt(100)
```

## ln

Return the natural log of the argument.

-----

### Synopsis

val = **ln**(*someval*)

-----

### Description

**ln** returns the natural log of its argument.

-----

### Arguments

  - *someval*  
    Any positive number, floating point or integer

-----

### Examples

```cpp
lval = ln(3.414)
```
